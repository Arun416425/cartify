<h1>🚀 Redux E-Commerce App</h1>

<h2>🛠️ Tech Stack</h2>
<li>React.js: Used for building a modular, component-based user interface.</li>

<li>Axios: Utilized to fetch data from external APIs and handle asynchronous logic.</li>

<li>Redux Toolkit: Implemented for centralized, predictable state management.</li>

<li>Custom CSS: Handcrafted styles utilizing Flexbox and Grid for a unique design.</li>

<li>Local Storage API: Integrated to ensure data persistence across browser sessions.</li>

<li>Vercel: Deployed for fast, reliable hosting and CI/CD integration.</li>

<h2>✨ Key Features</h2>
<li>Global State Management: Utilizes Redux to eliminate prop drilling and maintain a single source of truth.</li>

<li>Data Persistence: Automatically syncs the Redux state with browser Local Storage for seamless user experiences.</li>

<li>Custom Styling: Fully custom CSS implementation to demonstrate mastery over layout and design without external frameworks.</li>

<li>Responsive Design: Mobile-first architecture ensuring full functionality across mobile, tablet, and desktop devices.</li>

<li>Optimized Workflow: Efficient handling of state updates to ensure smooth performance.</li>

<h2>🏗️ Technical Architecture</h2>
<li>Store Configuration: Centralized store setup using Redux Toolkit’s configureStore.</li>

<li>Slices: Data organized into feature-based slices for maintainability.</li>

<li>Middleware: Custom logic to trigger Local Storage updates whenever the state changes.</li>

<li>Responsive Breakpoints: Strategic use of media queries to adapt the UI dynamically.</li>
